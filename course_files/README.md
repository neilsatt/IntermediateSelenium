# Installation

* `npm install -g mocha`
* Install one or more browser drivers:
  * `npm install -g chromedriver`
  * `npm install -g geckodriver` (for Firefox)
  * etc.
* Clone the repo.
* `cd` into the clone directory.
* Run `npm install`

# Running samples

* `cd` into a subdirectory.
* Run the `mocha` command.
